What is this?

Some time in mid-2025 I came across a really cool PDF file distributed by the National Association of Student Financial Aid Administrators: The NASFAA Data Sharing Decision Tree. It's a graph which enables a financial aid administrator to quickly determine whether financial aid data can be released to a person or entity. The graph encodes rules defined in the Higher Education Act (HEA), the Family Educational Rights and Privacy Act (FERPA), and the Privacy Act.

Being a software engineer by trade, and a computer programmer by avocation, it seemed natural and obvious to feed the PDF file into a Large Language Model (ChatGPT) to see if it could derive the rule set for the graph presented in the PDF.

To make what turned into a long (and growing) story short: It could not.

However, LLM could do a lot with the PDF:

  • Parse out the text with around 95% accuracy;
  • Determine about 75% of the rule transitions correctly;
  • Create a disclosure walkthrough;
  • Create a disclosure quiz, test your knowledge!
  • Build both terminal applications and web applications for disclosure and quiz;
  • Track development time.

That's pretty good!
